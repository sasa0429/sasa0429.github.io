变量

获取元素

document.getElementById(‘id’)

在当前文档中，通过id获取元素。

  • 只能通过id方式来获取。
  • 如果获取一个不存在的元素，得到结果为null。
  • 当遇到第一个符合条件的元素，就返回第一个元素。
  • 只有document下有，id是当前页面唯一的

document.querySelector(‘#id’)

通过元素选择器来获取，（选择器-css使用的选择器），id,class…

  • 如果获取一个不存在的元素，得到结果为null。
  • 当遇到第一个符合条件的元素，就返回第一个元素。
  • document / 指定父级下进行查找
  • IE8以下不可以使用

document.querySelectorAll(‘#id’)

通过元素选择器来获取，（选择器-css使用的选择器），id,class…

  • 获取到的是一组元素
  • 不能直接给一组元素加事件

    方括号可以填入需要计算的东西，或者不符合标识符规范的

字符串

由一对双引号 或 单引号抱起来的0个或多个字符组成的串

加号的作用

  • 加法运算：加号左右两侧必须是数字
  • 字符串拼接：作为连接符使用，将加号左右两边的内容拼接到一起

typeof 检测数据是不是字符串

注意：

当加号的左右任何一侧，有字符串，那么作为连接符使用

如果两边都是数字，作为加法运算使用

+=的使用

A = A + B ==> A += B

这里需要额外注意，字符串拼接的情况下，A 和 B 的顺序是不能颠倒的

if语句语法

  • 只有“如果”
  • “如果…否则…”

注意：

1.判断条件可以有多个

比较运算符 和 逻辑运算符

  • 比较运算符

大于 >

小于 <

等于 ==

  • 逻辑运算符

&& 与：同时满足才true

|| 或 : 其中一个满足就true

！ 非：取反

A && B：如果左侧的结果为true，则返回右侧的结果。如果左侧为false，直接返回左侧。

A || B：如果左侧的结果为true，则返回左侧的结果。否则返回右侧结果。

递增和递减

  • 递增（ ++ ）：每次为操作数增加1，并返回一个数值
  • 递减（ – ）：每次为操作数减少1，并返回一个数值

以上的三句话表达的都是一样的意思，但明显可以看到最后这句最简单
